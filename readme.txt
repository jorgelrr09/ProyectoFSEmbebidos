Fundamentos de Sistema Embebidos
Semestre 2021-1
Grupo: 2
Proyecto: Centro Multimedia

Descripción: Centro multimedia para la reproducción de películas, videos, música y fotografías. El sistema embebido descrito realiza las siguientes funciones:

Menú al encendido que permitirá elegir entre servicios en línea para video (Netflix), música (Spotify) o reproducir el medio extraíble (Memoria USB).
Cuando se inserte un medio extraíble (memoria USB) con fotografía, estas serán reproducidas en modo presentación.
Cuando se inserte un medio extraíble (memoria USB) con música, las pistas serán reproducidas una tras otra.
Cuando se inserte un medio extraíble (memoria USB) con videos, los videos serán reproducidos uno tras otro.
Cuando se inserte un medio extraíble (memoria USB) con contenido mixto el apartado deberá preguntar qué acción tomar.
